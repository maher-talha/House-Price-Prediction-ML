# House Price Prediction using Machine Learning

This project predicts house prices using a basic Machine Learning model in Python.

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- NumPy

## Model
Linear Regression is used to train the model and predict house prices.

## Evaluation Metrics
The model performance is evaluated using:
- MAE (Mean Absolute Error)
- MSE (Mean Squared Error)
- RMSE (Root Mean Squared Error)
- R² Score

This project demonstrates the basic concept of regression and prediction in machine learning.
